# NLS Folder - Organized Localization Files

This folder contains organized versions of the `package.nls.*.json` files that are required for extension localization in the VS Code Marketplace.

## Structure

- `package.nls.json` - Default language (English)
- `package.nls.{locale}.json` - Files for each supported language

## How It Works

### Automatic Generation

The files in this folder are automatically generated by the script `scripts/create-package-nls.js` based on the source files in `l10n/bundle.l10n.*.json`.

### Dual Localization

For organization and compatibility reasons, the `package.nls.*` files are maintained in two locations:

1. **Project root** (`/package.nls.*.json`)
   - Required for VS Code Marketplace to detect and show translations
   - Ignored by Git (auto-generated)
   - **DO NOT edit manually**

2. **This folder** (`/nls/package.nls.*.json`)  
   - Organized versions for development
   - Committed to Git for version control
   - Facilitate maintenance and review

## Commands

```bash
# Generate/update localization files
npm run generate:package-nls

# Or run directly
node scripts/create-package-nls.js
```

## Important

- **Never edit these files directly**
- Always edit the source files in `l10n/bundle.l10n.*.json`  
- Run the generation script after changes to source files
- Root files are automatically recreated during build
